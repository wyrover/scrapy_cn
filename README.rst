Scrapy project as a chinese tutorial 
====================================
scrapy中文学习项目

更多scrapy相关资料参考：<www.addbook.cn>

以抓取豆瓣图书中的编程图书为例：
地址：<http://book.douban.com/tag/Programming>

requirements::

    scrapy >= 0.22
    python == 2.7

Install::

    git clone https://github.com/addwork/scrapy_cn.git

Usage::

    $cd scrapy_cn
    $scrapy list
    $scrapy crawl onepage       #
    $scrapy crawl morepage      #
